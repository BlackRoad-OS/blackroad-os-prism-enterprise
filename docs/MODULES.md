# Modules Overview

| Domain        | Description                                               | Bots                               |
|---------------|-----------------------------------------------------------|------------------------------------|
| Finance       | Treasury, close, accounts payable/receivable automation.  | Treasury-BOT, Close-BOT            |
| Supply Chain  | Sales & operations planning, inventory, logistics.        | S&OP-BOT                           |
| Compliance    | Policy enforcement and approval workflows.                | Policy module (no direct bot)      |
| Lineage       | Data provenance, task/bot linkage.                        | Part of orchestrator core          |
| Security      | PII redaction and audit log signing.                      | Redaction service, audit trail     |

Refer to `docs/modules/` for detailed module guides.
